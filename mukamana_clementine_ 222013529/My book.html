    • Tables

      1. category Table

        2. The Category table stores information about product categories. It can be used to organize products into different categories and manage category details such as name and description..


      2. order Table

        3. The Order table tracks customer orders placed through the system. It can be used to manage order details such as order date, total amount, and order status (e.g., pending, shipped, delivered).


      3. customer Table

        4. The Customer table contains information about customers who use the system to place orders. It can be used to manage customer details such as name, email, phone number, and shipping address..


      4. supplier Table

        5. The Supplier table contains information about the suppliers who provide products to the system. It can be used to manage supplier details such as name, contact person, email, and phone number..


      5. payment Table

        6. The Payment table tracks payments made for orders. It can be used to manage payment details such as payment date, amount paid, and payment method (e.g., credit card, PayPal)..


      6. Products Tables

        7. The Product table stores information about each product available for sale. It can be used to manage product details such as name, description, price, quantity in stock, and the supplier and category to which the product belongs..
